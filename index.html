Não foi possível carregar o relatório

A URL configurada pode estar bloqueando a exibição em iframe.
Certifique-se de usar a URL gerada por "Publicar na Web" no Power BI Service.

Como gerar a URL correta:
1. Acesse app.powerbi.com e abra o relatório
2. Clique em Arquivo → Publicar na Web
3. Clique em "Criar código de inserção"
4. Copie a URL que começa com https://app.powerbi.com/view?r=...
5. Cole essa URL no campo do relatório no portal
